Файлы
* git status		- все изменения файлов репозиторя
* git add           - фиксация файлов ( . -фиксация всех файлов, git add <file> - один файл )
* git commit        - коммит файлов (-m '<comment>' - добавление комментария к коммиту)
* git push          - отправить на сервер
* git fetch         - связывается с удалённым репозиторием и забирает из него все изменения, которых у вас пока нет и сохраняет их локально
* git merge         - слить изменения с текущей веткой в проект
* git pull          - забрать изменения из указанного удалённого репозитория, а затем слить их с текущей веткой
* git init          - создание git-репозитория из данного каталога
