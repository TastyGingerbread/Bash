Файлы
* touch <file>      - создание файла 

Каталоги
* ls 				- содержвние каталога (-i размеры )
* mkdir name		- создание каталога.
* rmdir name		- удаление каталога.
* cp src dest		- копирование файла с затиранием уже существующего.
* mv src dest		- копирование/переименование файла и каталога с затиранием уже существующего файла или вставкой каталога в каталог.
* pwd 				- текущее расположение в каталогаз

Сеть 
* ssh 				-ssh - соединение (ssh <ip> -p <port> -l <user>)

Время
* date				- дата.
* cal				- календарь.
